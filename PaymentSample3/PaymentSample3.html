payment sample 1

This payment methos sample is a sample PHP implementation of a payment processor.

It does not really connecting to anything, it just shows a page where the user can press a button to make the order fail or succeed.

There is also a processing button. When this is pressed the order is set to success only at the next time the hourly cron is called. (which can be 1 minute or 59 minutes from the time the order was created).

This payment sample serves purposes

1. As a sample on how to implement a payment provider for IncraShop

2. As a test implementation to directly make an order fail or succeed when trying to order

Do not install this unmodified in the life environment as customers can buy products without actually paying them.
